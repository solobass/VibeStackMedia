# The Creative Vibe Stack & Prompt Coding Stack

## Creative Vibe Stack

The Creative Vibe Stack maps tools across the creator pipeline:

1. **Inspiration** - Tools for ideation and concept development
2. **Generation** - AI tools that create content
3. **Refinement** - Tools for editing and enhancement
4. **Assembly** - Tools for combining and structuring content
5. **Distribution** - Tools for publishing and sharing

## Prompt Coding Stack

The Prompt Coding Stack defines prompt complexity levels:

- **Level 1**: Basic single prompts
- **Level 2**: Structured prompts with parameters
- **Level 3**: Multi-step prompts
- **Level 4**: Chained prompts with context
- **Level 5**: Advanced workflows with conditional logic
